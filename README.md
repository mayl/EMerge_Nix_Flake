# EMerge Nix Flake

This flake builds EMerge and a devshell for running it. To enter the devshell
run:

```
nix develop github:mayl/EMerge_Nix_Flake
```

This doesn't support all the solvers EMerge does, but it does have `UMFPACK`.

In the interest of full disclosure, this flake was shamelessly vibe-coded with `claude-code`.
